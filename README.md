Console Colors
==============

Console Colors it's a very small script (30 lines) that adds colors to the node console.log extending the string prototype.  

###Styles supported:  
- underline
- inverse
- white

###Foreground colors:
- white
- grey
- black
- blue
- cyan
- green
- magenta
- red
- yellow

###Background colors:
- whiteBG
- blackBG
- blueBG
- cyanBG
- greenBG
- magentaBG
- redBG
- yellowBG
